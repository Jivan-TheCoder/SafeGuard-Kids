# SafeGuard Kids - Parental Control Landing Page

A premium, high-conversion landing page designed for an advanced parental control and child safety monitoring application. The design focuses on building trust with parents through high-fidelity aesthetics, clear messaging, and modern UI patterns.

## 🌟 Key Features

* **Premium UI/UX**: Built with a "pro" aesthetic utilizing vibrant indigo gradients, glassmorphism effects (backdrop-filters), and subtle hover animations to maximize user engagement.
* **Responsive Design**: Fully responsive layout that gracefully scales from large desktop monitors down to mobile devices, ensuring a seamless experience across all platforms.
* **Sticky Header**: A modern sticky title bar with glassmorphism that keeps navigation accessible during scrolling.
* **"Protect What Matters Most" Section**: A perfectly calibrated, full-width section with accurate copy, crisp checkmark icons, and a high-trust stats board.
* **Pixel-Perfect Alignment**: Exacting typography and spacing logic applied to ensure a polished, trustworthy presentation.
* **Shell-Based Verification Built-In**: Includes a strict `check_site.ps1` PowerShell script used to validate the HTML and CSS structural integrity without requiring browser intervention.

## 🛠️ Technology Stack

* **HTML5**: Semantic and clean HTML structure.
* **Vanilla CSS3**: Advanced CSS styling without relying on heavy frameworks (utilizing Flexbox, Grid, Custom Properties, and Backdrop Filters).
* **SVG Iconography**: Lightweight, resolution-independent icons embedded directly for optimal performance.

## 🚀 Getting Started

To view the landing page, simply open the `index.html` file in any modern web browser. No complex build tools or dependencies are required.

```bash
# Clone the repository
git clone https://github.com/Jivanlal786/Tracker-Website.git

# Open index.html in your browser
start index.html
```
